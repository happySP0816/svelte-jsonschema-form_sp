## Handlers

By providing `onSubmit` handler, you can achieve the form data validation and get a snapshot of the current form data
(if valid, otherwise the `onSubmitError` handler will be triggered with the errors).

<LinkCard title="Handlers API reference" href="../../api-reference/handlers/" />
